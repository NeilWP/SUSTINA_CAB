## 1. Purpose

Defines how data is **classified, handled, protected, and minimised** based on privacy impact.

---

## 2. Data Classification Levels

| Classification | Description |
|---------------|-------------|
| Public | No personal data |
| Internal | Non-sensitive operational data |
| Personal | Identifiable personal data |
| Special Category | GDPR Article 9 data |

## 3. Handling Rules

| Classification | Handling Requirements |
|---------------|----------------------|
| Personal | Lawful basis, minimisation, access control |
| Special Category | Explicit consent or statutory basis |
