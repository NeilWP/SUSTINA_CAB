## 1. Description & Scope

Mapping between CorporateEntity and **legal/regulatory industry classifications** (e.g. NACE).

---

## 2. Master / Reference / Mapping Role

| Layer | Role |
|-----|-----|
| **Master** | Z-01 CorporateEntity |
| **Reference** | NACE_Activity_Master |
| **Mapping** | Z-01.05 Industry Map |

---

## 3. Table Definition

| Column | Type | Notes |
|------|------|------|
| `Entity_Industry_Id` | INT IDENTITY | Primary key |
| `CorporateEntityId` | INT | Master link |
| `NACE_Code` | NVARCHAR(10) | Reference |
| `IsPrimary` | BIT | One per entity |
| `IsActive` | BIT | Soft deactivate |
| `ValidFromUtc` | DATETIME2(3) | Optional |
| `ValidToUtc` | DATETIME2(3) | Optional |
| `CreatedBySiteUserGuid` | UNIQUEIDENTIFIER | Audit |
| `CreatedAtUtc` | DATETIME2(3) | Audit |
| `ModifiedBySiteUserGuid` | UNIQUEIDENTIFIER | Audit |
| `ModifiedAtUtc` | DATETIME2(3) | Audit |

---

## 4. Business Rules

- At most one primary industry per entity
- All codes must exist in reference master
- Temporal validity supports regulatory reporting
