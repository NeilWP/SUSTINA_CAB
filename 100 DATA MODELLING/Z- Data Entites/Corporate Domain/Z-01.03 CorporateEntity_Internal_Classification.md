## 1. Description & Scope

Mapping table assigning **internal organisational classifications** to CorporateEntities
(e.g. DIVISION, REGION, SEGMENT).

---

## 2. Master / Reference / Mapping Role

| Layer | Role |
|-----|-----|
| **Master** | Z-01 CorporateEntity |
| **Reference** | Z-01.04 Classification_Type_Master |
| **Mapping** | Z-01.03 Internal Classification |

---

## 3. Table Definition

| Column | Type | Notes |
|------|------|------|
| `Classification_ID` | INT IDENTITY | Primary key |
| `CorporateEntityId` | INT | Master link |
| `Classification_Type_Code` | NVARCHAR(50) | Reference code |
| `Classification_Value` | NVARCHAR(250) | Applied value |
| `IsActive` | BIT | Soft deactivate |
| `ValidFromUtc` | DATETIME2(3) | Optional |
| `ValidToUtc` | DATETIME2(3) | Optional |
| `CreatedBySiteUserGuid` | UNIQUEIDENTIFIER | Audit |
| `CreatedAtUtc` | DATETIME2(3) | Audit |
| `ModifiedBySiteUserGuid` | UNIQUEIDENTIFIER | Audit |
| `ModifiedAtUtc` | DATETIME2(3) | Audit |

---

## 4. Business Rules

- Multiple classifications per entity allowed
- Only active reference codes permitted
- Temporal fields enable as-at reporting
